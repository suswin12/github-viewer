# GitHub Viewer

A modern web application that allows users to search, compare, and explore GitHub profiles using the GitHub API. The application displays detailed user information, repositories, followers, and profile statistics with a clean and responsive interface.

## Features

- Search GitHub Users
- Compare Multiple GitHub Profiles
- View User Profile Details
- Display Public Repositories
- View Followers & Following Count
- Profile Statistics Display
- Responsive User Interface
- Fast API Data Fetching
- GitHub API Integration
- Clean and Simple Design
- Real-Time Data Display
- Error Handling for Invalid Users

## Tech Stack

- React.js
- JavaScript
- CSS
- HTML
- GitHub API

## Purpose

To practice React development, API integration, state management, and responsive UI design by building a real-world GitHub profile viewer and comparison application.
